■概要

Outlookのメモを付箋紙にします。「Outlookのメモは付箋紙として使い
づらい」という場合に便利です。

■インストール

インストーラを実行します。 WindowsVista/7の場合は、インストール
の途中でUACの確認ダイアログが表示されますので、管理者権限を持つ
ユーザを入力してください。

次に、デスクトップに作成された[付箋紙を表示]のアイコンを実行しま
す。実行すると次のダイアログが表示されます。

[OK]を押すと初期設定が終わります。Outlookを起動していた場合には、
Outlookが終了されます。

■アンインストール

アンインストーラを実行します。 スタートメニューから[Outlookで付
箋紙のアンインストール]を実行してください。 もしくは、[プログラ
ムの追加と削除]から[tsoft Outlookで付箋紙]を削除してください。

WindowsVista/7の場合は、アンインストールの途中でUACの確認ダイア
ログが表示されますので、管理者権限を持つユーザを入力してください。

■使い方

基本的な使い方です。

≪付箋紙の追加・編集≫

・新しい付箋紙を作成したい
　⇒[新しい付箋紙]のアイコンを実行すると、真っ新な付箋紙が作成さ
　　れます。

・クリップボードの内容で新しい付箋紙を作成したい
　⇒[クリップボードから付箋紙]のアイコンを実行すると、クリップボ
　　ードの内容で新しい付箋紙が作成されます。

　いずれの場合でも空の付箋紙がある場合には、削除します。

≪付箋紙の表示≫

・ログインしたら付箋紙を表示したい
　⇒ログインすると自動的に付箋紙が表示されます。スタートアップフ
　　ォルダに[付箋紙の表示]が追加されています。

・付箋紙の表示/非表示を切り替えたい
　⇒[付箋紙の表示]のアイコンを実行すると、付箋紙が全表示/全非表
　　示されます。

≪付箋紙の管理≫

・付箋紙の一覧を確認したい
　⇒スタートメニューから[Outlookメモの管理]を実行すると、Outlook
　　のメモの画面が表示されます。[分類項目]に"付箋紙"と設定された
　　メモが付箋紙です。

・付箋紙をバックアップしたい
　⇒Outlookのメモ画面から行ってください。

■高度な使い方

もう少し高度な使い方です。

・メモを付箋紙に変更したい
　⇒[Outlookメモの管理]から対象のメモを選択し、右クリックメニュ
　　ーから[分類]の"付箋紙"を選択します。

・付箋紙ではなくメモに変更したい
　⇒[Outlookメモの管理]から対象のメモを選択し、右クリックメニュ
　　ーから[分類]の"付箋紙"を外します。

・[付箋紙の表示]をしても、 決して消えない付箋紙にしたい
　⇒[Outlookメモの管理]から対象のメモを選択し、右クリックメニュ
　　ーから[分類]の"付箋紙/表示"を選択します。

・付箋紙を作成したときに、特定の分類項目にしたい
・付箋紙を表示するときに、特定の分類項目だけ表示したい
　⇒ショートカットをコピーします。コピーしたショートカットのプロ
　　パティを開きます。

　　　　// 修正前
　　　　outlook-note.vbs /new
　　　　outlook-note.vbs

　　リンク先に下線部を追加します。この場合は分類項目が"すぐにや
　　る"の場合です。

　　　　// 修正後 - 分類項目の"すぐにやる"を追加
　　　　outlook-note.vbs /new /tag:すぐにやる
　　　　outlook-note.vbs /tag:すぐにやる

　　分類項目はカンマで区切って2個以上並べることができます。

　　　　// 修正後 - 分類項目として、"すぐにやる"、
　　　　// "今日やる", "重要"の3つを指定している場合です。

　　　　outlook-note.vbs /new /tag:すぐにやる,今日やる,重要
　　　　outlook-note.vbs /tag:すぐにやる,今日やる,重要
