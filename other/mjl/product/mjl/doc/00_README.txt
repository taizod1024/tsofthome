## MJL - MaJhong Logger service

■ はじめに

1.MJL とは

...麻雀の対戦結果を Web 上で入力し、Web 上で公開するサービスです。

Web が事実上の情報共有の基盤システムになりながら、そこで提供される情報
は正規化されていません。正規化されていないが故に、共有はされながらも再
利用・統計・分析が困難です。麻雀の対戦結果についてもまったく同じです。
麻雀に勝つのには実力・運のどちらがより必要とされているのかは分かりませ
んが、少なくとも自分の過去の対戦結果がどうであったのかを振り返ることは
自分の未来において少しでも勝率を高めることに有益ではないでしょうか。対
戦相手と共に過去の対戦結果を振り返り切磋琢磨するためのサービス、それが
MJL です。

2.こんな方に

MJL はこんな方々の利用を想定しています。

・比較的身内で麻雀を打っている方々
・時々麻雀大会がある方々
・勝ったり負けたりしているようだが実際どうだか分からない方々

麻雀の結果を集計するのが面倒な方々は是非使ってみてください。

■ 機能

1.Web 上で麻雀の対戦記録をつけることができます。

2.対戦結果を記録できます。

2-1.対戦記録は 1)日付、2)対局番号、3)半荘番号で識別されます。
2006-04-17_1-3
2-2.対戦記録は 1)日付別、2)対局番号別にタイトルをつけることができます。
2006 年度 1Q 大会 一回戦第一試合
2-3.対戦記録は 1)日付別、2)対局番号別、3)半荘番号別にコメントをつけるこ
とができます。
T の三倍満が炸裂
2-4.コメントには MJL 独自の記法により麻雀牌をイメージ表示することができ
ます。
ワンズ：一二三四五六七八九
ピンズ：１２３４５６７８９もしくは ①②③④⑤⑥⑦⑧⑨
ソウズ：ⅠⅡⅢⅣⅤⅥⅦⅧⅨ
字牌　：白発中東南西北
※ピンズだけ二つあるのは丸付き数字が機種依存文字であるが、分かり
やすさのため捨てがたいため。

3.対戦結果を参照できます。

3-1.対戦記録は直近の 1 ヶ月分が自動的に表示されます。
3-2.対戦記録はカレンダーから選択できます。
3-3.対戦結果を参照中に、対戦結果を新たに登録できます。
3-4.対戦結果を参照中に、対戦結果を集計することができます。
3-5.各対戦記録および分析結果は個別の URL で参照されます。
http://サーバ名/mjl/log/
http://サーバ名/mjl/log/mmjl0010.aspx
http://サーバ名/mjl/log/mmjl0010.aspx?gr=グループ&y=2006&m=12&d=04&gm=1&s=s1&m=ユーザ 1,ユーザ 2,...

4.対戦結果を集計できます。

4-1.対象期間は以下のとおりです。
全期間、年度、上期、下期、四半期、上半期、下半期、月
4-2.集計項目は以下のとおりです。
平均得点、合計得点、最高得点、最低得点
平均順位、各順位の回数

5.対戦結果を分析できます。

5-1.メンバを指定して分析します。対象は重複しない一人以上のメンバです。
5-2.対象期間は以下のとおりです。
全期間、年度、上期、下期、四半期、上半期、下半期、月
5-3.分析内容は以下のとおりです。
対戦相手の平均得点(差)、合計得点、最高得点、最低得点
対戦相手の平均順位(差)、各順位の回数

9.補助機能は以下のとおりです。

9-1.対戦メンバは登録できます。また、メンバ名は後から修正でき、以前の入
力内容に反映できます。
3-2.登録した内容は全て Web 上に公開され共有されます。情報はグループごと
に区切られます。そのグループに登録する場合には、グループ開設時に登
録した秘密の質問に回答する必要があります。
3-3.参加回数の多いメンバ、少ないメンバは色づけして表示できます。
3-4.登録したデータは全てエクスポートできます。
3-5.既に有るデータをインポートすることができます。

■ 利用規定

・MJL が提供する登録内容、集計結果、分析結果を含む全ての機能および性能に
ついて何も保証も保障もしません。
－登録したが MJL のバグで破壊された場合や
－分析結果が常に間違っている場合
※善意でやってるから間違ってても文句言うなということ

・MJL を利用することで如何なる被害を受けた場合でも補償はしません。
－MJL がクラックされて利用者の個人情報が漏洩した場合
－MJL がクラックされて利用者の端末がウィルスに汚染した場合
※善意でやってるから勘弁してくれということ。

・MJL の使用に際して画面表示された内容は、それが利用者の対戦結果であった
としても著作権は MJL 運営側が保有するものとします。現時点では著作権の保
有は、MJL 運営側の存続に危機を及ぼす場合だけに限定して行使するものとし
ます。
※使ってもらって嬉しいけど入力行為により著作権は発生しないとさせてくだ
さい。面倒くさいから。

・登録されたデータ自体は MJL 運営側が所有するものとします。ただし、エクス
ポートされたデータについては利用者が所有するものとします。
※どこから利用者のものになるかというと、エクスポートしたデータだけ

・MJL は現時点では無料ですが、今後継続して無料であることは保証しません。
ただし、有料化する場合でも、有料化宣言を遡っての課金はしません。
※別に有料化する予定はないけど、広告が付いて価値がついたらら売却もあり

・MJL の存続に危機を及ぼすと MJL 運営側が判断した場合には、利用を停止させて
いただきます。その際に事前通知および理由の開示は不要とします。
※善処するけどトラブルは面倒だから運営停止とします。

・ソースコードおよび関連物の公開は行いません。

・MJL のサービスの内容は利用者の承諾無しに変更されます。利用者はそのサー
ビス内容を承諾するものとします。
※不服を申し立てるのは OK だけど

・この利用規定は利用者の承諾無しに変更されます。利用者はその利用規定を承
諾するものとします。
※不服を申し立てるのは OK だけど

・上記に反しない限り、できる限り利用者が有料にならないように MJL 運営側は
善処します。
